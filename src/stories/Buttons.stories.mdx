# Buttons

Buttons are used to trigger actions within an application.

## Usage

Buttons are used to trigger internal actions within your web applications.

## Primary Buttons

Primary buttons are used as a call to action like submitting the form and indicate the most important action on a page.

<Preview withToolbar>
  <Story name="primary">
    <PrimaryButton onClick={action("primary-button-clicked")}>
      Buy Now
    </PrimaryButton>
  </Story>
</Preview>

## Secondary Buttons

Primary buttons are used as a call to action and indicate the important actions on a page.

<Preview withToolbar>
  <Story name="secondary">
    <SecondaryButton onClick={action("secondary-button-clicked")}>
      Back
    </SecondaryButton>
  </Story>
</Preview>
